This plugin allows you to add a slideshow with a shortcode anywhere on the site

== Description ==

Quickly display a slideshow on your website.

Just upload the images, add link url, alt text, etc and arrange them the way you need by drag and drop.

**Shortcode Example:**
`[myslideshow]`

== Features ==
- **Number of slides to show:** The number of slides to show in container. Default: "1"
	`slides="1"`

- **Show title** If activated, it shows the title at the bottom of the image. Default: "1"
	`title="1"`

- **Show arrows** If activated, it shows scroll arrows on the sides of the image. Default: "1"
	`arrows="1"`

- **Show dots** If activated, it shows dots below the image for the displacement of slides. Default: "1"
	`dots="1"`
